用 户 咨 询 订 单 分 几 个 快 递 发 货 。 用 户 询 问 收 到 两 个 （ 浴 霸 和 一 个 灯 ） 还 少 了 一 个 大 灯 。
用 户 询 问 今 后 商 品 损 坏 去 处 维 修 ， 可 否 更 换 电 话 号 码 。 用 户 询 问 差 价 是 都 已 经 退 还 。
用 户 咨 询 快 递 什 么 时 候 到 ， 以 及 再 投 是 什 么 意 思 。
用 户 询 问 是 否 可 以 [ 数 字 ] 号 以 后 发 。 用 户 询 问 在 哪 里 填 写 预 计 发 货 时 间 。 用 户 表 示 快 递 服 务 里 面 好 像 没 有 想 要 的 时 间 。 用 户 询 问 什 么 时 候 发 货 。 用 户 询 问 是 否 是 送 货 上 门 。 用 户 询 问 下 了 单 但 是 没 有 付 款 是 否 可 以 改 地 址 。
用 户 询 问 是 否 订 单 完 成 后 客 服 会 帮 助 保 价 以 及 没 付 款 的 订 单 能 否 取 消 。 用 户 询 问 明 天 到 期 了 是 否 有 关 系 。
用 户 询 问 凑 单 超 过 后 为 何 农 夫 山 泉 还 收 运 费 。 用 户 询 问 能 否 取 消 订 单 。
用 户 询 问 益 生 菌 和 益 生 元 一 样 吗 。 用 户 询 问 德 版 和 荷 兰 版 有 什 么 区 别 。 用 户 询 问 什 么 时 候 有 货 。
用 户 反 映 他 买 的 电 动 牙 刷 少 给 了 一 个 牙 刷 头 ， 并 反 映 他 打 电 话 说 补 发 到 现 在 也 没 消 息 ， 都 [ 数 字 ] 天 了 。 用 户 询 问 什 么 升 级 。 用 户 询 问 现 在 为 什 么 不 能 告 诉 ， 都 [ 数 字 ] 天 了 。
用 户 购 买 的 京 东 保 修 服 务 跟 品 牌 商 的 保 修 服 务 重 合 了 两 年 ， 用 户 希 望 客 服 帮 忙 退 京 东 的 保 修 服 务 。
商 家 反 映 男 装 招 商 电 话 打 不 通 并 询 问 花 花 公 子 品 牌 能 否 入 住 。 商 家 询 问 转 移 店 铺 类 目 如 何 操 作 。 商 家 询 问 是 否 可 以 新 增 类 目 ， 了 解 一 下 是 否 需 要 增 加 保 证 金 。
用 户 询 问 订 单 地 址 填 错 并 且 已 经 提 交 退 款 申 请 还 是 否 可 以 退 款 ， 大 概 需 要 多 长 时 间 审 核 通 过 。 用 户 询 问 钱 是 否 直 接 反 到 礼 品 卡 里 。 用 户 询 问 一 个 订 单 被 拆 分 为 四 个 订 单 是 否 能 都 取 消 。
用 户 询 问 商 品 降 价 是 否 可 以 申 请 价 保 。
用 户 询 问 个 人 抬 头 发 票 可 否 补 开 增 值 税 发 票 。 用 户 询 问 企 业 抬 头 发 票 是 否 只 能 改 为 相 同 抬 头 的 增 值 税 发 票 。
用 户 购 买 商 品 缺 件 要 求 补 货 或 者 退 货 。 用 户 表 示 已 提 交 申 请 ， 询 问 多 久 可 以 审 核 结 束 。 用 户 询 问 是 否 会 按 照 约 定 时 间 送 货 。
用 户 想 知 道 订 单 什 么 时 候 能 送 到 并 询 问 订 购 的 空 调 ， 页 面 反 应 要 从 外 地 调 货 ， 是 什 么 情 况 。 用 户 询 问 能 否 保 证 [ 数 字 ] 号 到 货 。
用 户 希 望 次 日 收 到 货 物 。
用 户 表 示 货 到 商 品 降 价 ， 申 请 价 保 。 用 户 询 问 价 格 保 护 周 期 。 用 户 询 问 电 脑 端 在 哪 里 再 次 申 请 。
用 户 询 问 退 货 单 为 何 还 没 处 理 。 用 户 表 示 自 己 的 手 机 号 有 问 题 ， 并 提 供 了 新 的 联 系 方 式 。
用 户 催 问 商 品 什 么 时 候 能 送 到 。 用 户 询 问 为 什 么 还 要 再 等 一 天 。 用 户 询 问 大 概 几 点 能 送 到 。
用 户 咨 询 收 到 的 迷 你 音 响 为 什 么 没 有 内 存 卡 。
用 户 询 问 发 票 怎 样 修 改 税 号 。 用 户 询 问 需 提 供 哪 些 信 息 。
用 户 询 问 朋 友 代 付 订 单 退 款 退 到 哪 里 。 用 户 询 问 需 要 多 久 。
用 户 询 问 订 单 地 址 错 了 怎 么 修 改 。 用 户 询 问 为 什 么 有 另 一 个 城 市 的 订 单 无 法 退 款 。 用 户 询 问 券 会 不 会 退 ， 退 款 是 原 路 返 还 吗 ， 并 询 问 什 么 时 间 到 账 。
