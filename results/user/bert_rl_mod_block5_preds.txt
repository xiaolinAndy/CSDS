用 户 询 问 何 时 发 货 。 用 户 询 问 是 否 可 以 送 货 上 门 ， 是 否 可 以 修 改 地 址 。
用 户 询 问 锁 着 的 订 单 明 天 到 期 了 是 否 有 关 系 。 用 户 询 问 如 何 取 消 没 付 款 的 订 单 。
用 户 询 问 为 何 农 村 收 了 运 费 。
用 户 询 问 德 版 和 雪 版 的 区 别 。 用 户 询 问 商 品 无 货 ， 何 时 有 货 。
用 户 反 映 购 买 的 电 动 牙 刷 头 少 了 一 个 牙 刷 头 ， 补 发 到 现 在 也 没 消 息 。
用 户 想 要 退 保 修 的 服 务 。 用 户 表 示 京 东 保 修 三 年 和 品 牌 商 保 修 重 合 了 两 年 。
商 家 询 问 男 装 招 商 的 电 话 怎 么 打 了 都 没 人 接 。 商 家 询 问 是 否 可 以 增 加 保 证 金 。
用 户 询 问 订 单 地 址 写 错 怎 么 办 。 用 户 询 问 还 可 以 退 款 吗 。
用 户 表 示 买 的 打 印 纸 就 降 价 。
用 户 询 问 忘 记 开 增 票 后 面 是 否 可 以 补 开 增 票 。 用 户 询 问 电 子 发 票 抬 头 是 个 人 的 可 以 修 改 为 增 票 吗 。
用 户 询 问 审 核 时 间 。 用 户 询 问 明 天 上 午 取 货 ， 京 东 快 递 人 员 是 否 会 过 来 。
用 户 咨 询 订 购 的 三 台 空 调 要 从 外 地 调 货 的 原 因 。 用 户 询 问 京 东 什 么 时 候 送 货 。
用 户 希 望 明 天 早 上 收 货 。
用 户 因 商 品 降 价 希 望 价 格 保 护 。 用 户 询 问 价 格 保 护 周 期 。
用 户 询 问 为 什 么 退 货 单 还 没 有 处 理 。 用 户 表 示 商 品 到 现 在 看 到 这 个 附 件 都 没 动 ， 机 子 也 没 有 机 激 活 。
用 户 询 问 订 单 送 达 时 间 。
用 户 表 示 订 单 中 的 商 品 没 有 内 存 卡 。 用 户 询 问 另 一 个 订 单 号 。
用 户 询 问 在 换 公 司 的 税 号 如 何 修 改 。
用 户 询 问 朋 友 代 付 的 订 单 取 消 的 话 ， 退 款 退 到 哪 里 。 用 户 询 问 退 款 何 时 到 账 。
用 户 询 问 地 址 搞 错 怎 么 办 。 用 户 询 问 申 请 了 退 款 为 什 么 有 个 退 不 了 。
